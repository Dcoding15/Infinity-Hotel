# 🏨 Infinity Hotel Management System

Infinity Hotel is a comprehensive web-based platform designed to manage guest registrations and provide administrative tools for hotel staff. The project features a futuristic space-themed design and a responsive user interface.

## 🌟 Key Features

### 1. Guest Registration & Experience
* **Multi-Purpose Registration**: Guests can register for specific services like **Weddings**, **Meetings**, or standard **Stays**.
* **Interactive Support**: Includes a floating chat support system and a contact form on the home page for hands-on query resolution.
* **Confirmation System**: Users receive a visual success notification (via `popup.html`) upon completing their registration.

### 2. Staff & Administrative Portal
* **Secure Authentication**: A dedicated login portal for staff members to access sensitive data.
* **Customer Dashboard**: Staff can view a detailed list of all hotel customers, including:
    * Room Assignments
    * Contact Details
    * Requested Services (Wedding/Meeting/Stay)
* **Session Management**: Includes secure logout functionality to protect administrative data.

### 3. Modern UI/UX
* **Animated Design**: Features custom CSS animations, including a unique rocket launch 404 error page.
* **Responsive Layout**: Built with Bootstrap to ensure the site works on mobile, tablet, and desktop.
* **Visual Appeal**: A dark, "infinity" space theme using radial gradients and starfield backgrounds.

## 🛠️ Technical Stack

* **Frontend**: HTML5, CSS3, JavaScript (jQuery).
* **Framework**: Bootstrap 5.3 (Responsive Design).
* **Icons/Fonts**: Font Awesome, Google Fonts (Philosopher, Exo 2, Fira Mono).
* **Backend Ready**: Includes Django template tags (CSRF protection, loops) for server-side integration.

---

## 🚀 Getting Started

1.  **Guest Path**: Open `Home Page` -> Navigate to "Register" -> Fill details -> View success popup.
2.  **Staff Path**: Navigate to `Admin Login Page` -> Enter credentials -> View guest table in `Admin Page`.
